# AI Robotics: The Body & The Brain

This repository contains the companion code, hardware schematics, and resources for the **AI Robotics** curriculum designed by **Eng. Youssef Goher**.

This course bridges the chasm between low-level hardware abstraction (The Body) and high-level probabilistic cognitive processing (The Brain), utilizing the ESP32 microcontroller, MicroPython, and the Google Gemini API.

## 📚 Curriculum Overview

This curriculum is structured into two primary phases, totaling six lecture modules:

### Phase 1: The Body (Hardware & Robotics)

* **Lecture 1:** Setup & Python Essentials (ESP32, REPL, GPIO)
* **Lecture 2:** The Senses (HC-SR04, Ultrasonic Sonar, Data Acquisition)
* **Lecture 3:** Motion & Actuation (PWM, Servos, L298N H-Bridge, OOP Hardware Abstraction)

### Phase 2: The Brain (AI & Integration)

* **Lecture 4:** LLMs & Prompt Engineering (System Instructions, Negative Constraints)
* **Lecture 5:** The Bridge (REST APIs, HTTP/JSON, Networking, urequests)
* **Lecture 6:** Capstone (The Sentient Explorer — Full System Integration)

## 🛠 Technical Stack

This project is built entirely on the Python ecosystem, optimized for resource-constrained edge devices:

* **Microcontroller:** ESP32 (Xtensa LX6)
* **Firmware:** MicroPython
* **Logic Layer:** Google Gemini 1.5 Flash API
* **Communication:** REST/JSON
* **Components:** HC-SR04, L298N, Servo motors
